# THE BIN! 🗑️

> ปล่อยความกังวลของคุณให้เป็นหน้าที่เรา

แอประบายอารมณ์แบบ "ทิ้งแล้วหาย" — ไม่เก็บข้อมูล ไม่ถาม ไม่ตอบ ไม่จำ

## Features

- **3 input modes** — Type / Draw / Voice
- **4 effects** — Bin, Boom, Fade, Shhhh
- **Daily Boom** — streak + points system
- **PWA** — Add to Home Screen ได้เลย
- **Zero data retention** — content หายทันทีหลัง dispose

## Dev

```bash
npm install
npm run dev
```

## Deploy (Netlify)

```bash
npm run build
# drag dist/ to app.netlify.com
```

หรือ connect GitHub repo แล้ว auto deploy ทุกครั้งที่ push
